# R3A Wireless Earbuds Landing Page

A modern, responsive product landing page designed to showcase the R3A Wireless Earbuds. This project focuses on clean UI, smooth navigation, and clear calls to action.

## Features

* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **Sticky Navigation:** Easy access to different sections of the page.
* **Video Integration:** Embedded product demo to engage users.
* **Lead Generation:** A functional "Get Notified" form for user subscriptions.
* **Feature Highlights:** Highlighting technical specs like battery life and audio quality.

##  Built With

* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling for a premium, tech-focused aesthetic.

##  Preview

The page includes sections for:
1.  **Features:** Visual breakdown of the earbuds' capabilities.
2.  **Demo:** Video showcase of the product in action.
3.  **Pricing/Sign-up:** A contact form for potential customers.
